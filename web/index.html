🌤️ GeoWeather

A modern Android weather app, inspired by MeteoSwiss

Kotlin Jetpack Compose Material You Android API 26+ v1.3.8

About this preview

GeoWeather is a native Android application — it compiles to an .apk file and runs on an Android phone or emulator. It is not a web app, so there is nothing to render in the browser preview beyond this project landing page.

To build the app you need the Android SDK and JDK installed locally, then run ./gradlew assembleDebug. The compiled APK appears under app/build/outputs/apk/.

✨ Key features

🏙️

Multiple cities

Add and manage as many cities as you like.

🌡️

Unit switching

Toggle Celsius/Fahrenheit and km/h / mph.

📅

7-day forecast

Detailed weekly weather predictions.

⏰

Hourly forecast

Precise weather data hour by hour.

🎨

Material You

Dynamic colors from your wallpaper (Android 12+).

🔔

Notifications

Weather alerts and update reminders.

🧰 Build commands

./gradlew assembleDebug      # build a debug APK
./gradlew assembleRelease    # build an unsigned release APK
./gradlew installDebug       # install onto a connected device/emulator
